# Guilherme C. Mais — Portfolio

Portfolio pessoal desenvolvido com HTML5, CSS3 e JavaScript vanilla, apresentando projetos, habilidades técnicas e informações de contato.

## 🌐 Deploy

[www.guilhermecmais.com](https://www.guilhermecmais.com)

## 🛠️ Tecnologias

- HTML5 semântico
- CSS3 com variáveis customizadas e design tokens
- JavaScript vanilla (ES6+)
- Google Fonts — DM Sans + DM Serif Display

## ✨ Funcionalidades

- Cursor personalizado animado
- Efeito de digitação automática (typed text)
- Animações de entrada ao fazer scroll (reveal)
- Barras de progresso animadas nas skills
- Filtro de projetos por categoria
- Formulário de contato com validação
- Menu mobile responsivo
- Parallax no hero
- Botão voltar ao topo
- 100% responsivo para mobile, tablet e desktop
